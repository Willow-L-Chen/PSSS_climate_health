# Modelling Environmentally Mediated and Climate-Sensitive Infectious Syndromes in the Pacific Region (2011–2020)

This repository contains the R scripts used for data cleaning, environmental variable extraction, and mixed-effects negative binomial regression modelling for the manuscript:

> **Chen-Cao L., Cadavid Restrepo A., Skinner E.B., Kiani B., Sartorius B. (2026)**  
> *Modelling Environmentally Mediated and Climate-Sensitive Infectious Syndromes in the Pacific Region (2011–2020)*  

The repository is provided to enhance the **reproducibility and transparency** of the study’s analytical workflow.  
All scripts are structured for clarity and may be adapted to similar epidemiological analyses.

**Note:** The syndromic surveillance data analysed in this study are not publicly available due to confidentiality agreements with the Pacific Syndromic Surveillance System (PSSS).
